Use the following command to compile and run your MRSG simulation:

./compile_run <platform.xml> <deployplatform.xml> <yourconfig.conf>

Example: ./compile_run.sh platforms/mrsg_example32.xml platforms/d-mrsg_example32.xml example32config.conf

You may define the map output in the config file with mra_map_output <value>, the default value is calculated with the other data provided on the config file.
